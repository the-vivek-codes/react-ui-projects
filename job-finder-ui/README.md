# Job Finder UI

A modern and responsive Job Finder UI built using ReactJS and Vite.

## Preview

![Preview](./preview.png)

## Features

- Responsive job cards UI
- Reusable React components
- Dynamic rendering using `.map()`
- Interactive hover effects
- Modern flexbox layout
- Clean and minimal design

## Tech Stack

- ReactJS
- Vite
- CSS3
- JavaScript (ES6)

## Folder Structure

```bash
src
 ┣ assets
 ┃ ┗ bookmark-icon.svg
 ┣ components
 ┃ ┗ Card.jsx
 ┣ App.jsx
 ┣ index.css
 ┗ main.jsx
```

## Installation

```bash
npm install
npm run dev
```
